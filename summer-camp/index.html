What this is

a hands-on cohort, not a webinar

MVP Club is a community for people learning to build with AI — professionals who want to make something real, not just talk about the tools. AI Summer Camp is our four-Friday beginner cohort ($99): project-based, hands-on, and worked through together. You don't leave with notes. You leave having built something and shown it to the group.

Each session runs live on a shared screen — this website on the left as the lesson, Claude Code on the right as the workspace. We move through real workplace scenarios as a cohort, one Friday at a time, from a fuzzy idea to a finished demo.

You'll feel behind — and here's what we do about it

we hand you the durable skills, not the weekly churn

Let's be honest: everyone feels behind right now. This is a technology that changes by the week — a new model, a new tool, a new "you have to try this" every few days. That churn is real, and it's not going away.

But underneath the churn, something steadier is emerging: a set of core skills and durable mental models that hold even as the specific tools change. Knowing how to turn a fuzzy idea into a clear goal. How to scope and document before you build. How to direct an AI instead of being directed by it. These are what orient you in a moment of rapid disruption.

That's exactly what this camp develops and practices. We're not promising the churn will slow down. We're handing you the skills that outlast it — so the next new tool is something you pick up in an afternoon, not something that leaves you behind.

Set up Claude Code

do this once before Session 1 · pick the path that matches what you have

Every session uses Claude Code on the right side of your screen. Get it open and pointed at a folder before we start so we can dive straight into building. Pick whichever path matches your setup.

Path A — Terminal

Claude Code CLI in VS Code / Terminal / PowerShell.

  1. Create a folder on your Desktop named summer-camp (right-click → New Folder).
  2. Mac: right-click the folder → "New Terminal at Folder." · Windows: right-click inside it in Explorer → "Open in Terminal."
  3. Type claude and press Enter.
  4. If it's your first time, follow the prompts to log in. When you see the prompt, you're ready.

Path B — Desktop App

Claude Code Desktop App on Mac or Windows.

  1. Create a folder on your Desktop named summer-camp.
  2. Open the Desktop App → click the Code tab (not Chat or Cowork).
  3. Select Local → Select folder → pick summer-camp.
  4. Pick a model (Sonnet is fine to start).
Approving file writes: Terminal → press y. Desktop → click Accept on the diff.
Seeing your files: Terminal → use Finder/Explorer. Desktop → click the file icon in the sidebar.
One more thing: make sure you're in the AI Summer Camp cohort channel in the MVP Club community — that's where we share the session link, post prompts, and answer questions live each Friday.
